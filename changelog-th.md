## v1.3.1 - 2025-11-11

- แนะนำชื่อ Build ที่โหลดล่าสุดเมื่อบันทึก
  - จดจำชื่อ Build ที่เพิ่งโหลดมาแล้วเติมให้เป็นคำแนะนำเมื่อเปิดหน้าต่าง Save Build

- ต้องมีทั้งสกิลและไอเทมก่อนดำเนินการกับ Build
  - ปรับการตรวจสอบให้แสดงข้อผิดพลาด เฉพาะเมื่อทั้งสกิลและไอเทมว่างเปล่า เพื่อลดการแจ้งเตือนที่ไม่จำเป็น

- เพิ่มการเรียงลำดับ Build ที่บันทึกในหน้าต่าง Load Build
  - ผู้ใช้สามารถเรียง Build ที่บันทึกตามชื่อหรือวันที่ (ขึ้น/ลง) โดยมีปุ่มควบคุมใน UI ของหน้าต่าง

- เพิ่มการจัดลำดับด้วยการลาก-วางแบบ Manual
  - แนะนำโหมด "Manual" และไอคอนจับลากสำหรับจัดลำดับ Build ที่บันทึกใหม่ ลำดับที่กำหนดจะถูกบันทึกลง localStorage และมีฟีดแบ็กใน UI ขณะลาก
